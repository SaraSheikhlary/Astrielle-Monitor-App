# Astrielle AI: State Monitor 🚀

Welcome to the official repository for **Astrielle AI**. This application is a professional state monitoring dashboard powered by AI.

## ✨ Features
* **Live Voice Input:** Record and analyze audio in real-time.
* **Secure Monitoring:** Privacy-focused analysis with no data storage.
* **Instant Reports:** Downloadable analysis results.

## 🛠️ Technology
* [Streamlit](https://streamlit.io)
* Python
* Hosted on Streamlit Cloud
